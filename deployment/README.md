
# IITGEEPrep Deployment Guide (v12.17)

This guide explains how to deploy the IITGEEPrep application to a LAMP stack server (e.g., Hostinger, Bluehost, AWS EC2).

## 1. Source Folder Structure

After building the project and downloading the System Bundle, your source files will be organized as follows:

```
/ (Project Root)
├── dist/                     # Generated by `npm run build` (Frontend Code)
│   ├── assets/               # CSS/JS files
│   ├── index.html            # Entry point
│   └── ...
├── deployment/               # Backend & Config Files
│   ├── api/                  # PHP Backend Scripts
│   │   ├── config.php
│   │   ├── index.php
│   │   └── ...
│   ├── sql/                  # Database Schema
│   │   └── database.sql
│   └── seo/                  # Root Config Files
│       ├── .htaccess
│       ├── robots.txt
│       └── sitemap.xml
```

## 2. Server Destination Mapping

You must copy files from the **Source** folders to the specific **Destination** folders on your server.

| Component | Source Folder (Local) | Destination Folder (Server `public_html`) | Description |
| :--- | :--- | :--- | :--- |
| **Frontend** | `dist/*` (All contents) | `public_html/` | The compiled React application. `index.html` must be in the root. |
| **Backend API** | `deployment/api/*` | `public_html/api/` | PHP files handling logic. Create the `api` folder manually if needed. |
| **Database** | `deployment/sql/database.sql` | *Import via phpMyAdmin* | Do not copy to file manager. Import this into your MySQL DB. |
| **Config** | `deployment/seo/*` | `public_html/` | `.htaccess` handles routing. `robots.txt` handles SEO. |

## 3. Step-by-Step Instructions

### Step A: Database Setup
1. Create a new MySQL Database and User in your hosting panel.
2. Open **phpMyAdmin**.
3. Select your new database.
4. Go to **Import** tab -> Upload `deployment/sql/database.sql` -> Click **Go**.

### Step B: Backend Configuration
1. Open `deployment/api/config.php` locally.
2. Edit the file to add your database credentials:
   ```php
   $host = "localhost";
   $db_name = "YOUR_DB_NAME";
   $username = "YOUR_DB_USER";
   $password = "YOUR_DB_PASSWORD";
   ```
3. Save the file.

### Step C: File Upload
1. Upload the **contents** of `dist/` to `public_html/`.
2. Create a folder named `api` inside `public_html`.
3. Upload the **contents** of `deployment/api/` to `public_html/api/`.
4. Upload the **contents** of `deployment/seo/` to `public_html/`.

### Step D: Verification
1. Visit `https://your-domain.com/api`. You should see a JSON success message.
2. Visit `https://your-domain.com`. The app should load.
